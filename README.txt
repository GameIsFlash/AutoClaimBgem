Краткий гайд по работе с программой.

1. Скачиваем Python с официального сайта Python.org но версия дожна быть 3.7. Чтобы у вас не возникало никаких проблем. Так как некоторые библиотеки используемые в этом проекте можно было использовать только до версии Python 3.8, далее их поддержка была прекращена.
2. Скачиваем все файлы репозитория и всталяем их в любое удобное вам место (рекомендую сделать отдельную папку)
3. Файл Abi.json не трогаем, это файл который должен в точности повторять abi файл смарт-контракта BGEM
4. Прописываем в файле Account.json все свои аккаунты в формате "адрес кошелька":"закрытый ключ"
5. Дублируем адрес кошельков в adresses.txt без любых знаков препинания. Просто один адрес, одна строчка
6. Заходим на сайт https://polygonscan.com/ и смотрим там Med Gas Prise и вписываем это значение в 112 строку main.py вместо числа 210
7. Запускаем скрипт!
